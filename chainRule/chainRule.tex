\section{The Chain Rule}

Composition is an operation that is unique to functions and not to numbers.
The composition of two functions $f$ and $g$ is the function $f \circ g$ defined by
\[(f \circ g)(x) = f(g(x))\]
We can refer to $g$ as the ``inside'' function and $f$ as the ``outside'' function.  
Some examples of compositions are:
\begin{itemize}
\item $\sqrt{x^2 + 9}$ is the composition of the inside function $g(x) =x^2 + 9$ and the outside function $f(x) = \sqrt{x}$.
\item $e^{-x^2}$ is the composition of the inside function $g(x) = -x^2$ and the outside function $f(x) = e^x$.
\item $\ln(x^4 + 1)$ is the composition of the inside function $g(x) = x^4 + 1$ and the outside function $f(x) = \ln(x)$.
\item $\tan^3(x)$ is the composition of the inside function $g(x) = \tan(x)$ and the outside function $f(x) = x^3$.
\end{itemize}
Note that the inside function is the one that acts first on $x$ and the outside function is the one that acts last on $x$.
The chain rule is a formula for computing the derivative of a composition.
\begin{theorem}[Chain Rule] If $f(x)$ and $g(x)$ are differentiable functions, then
\[
\frac{d}{dx}\left[f(g(x))\right] = f'(g(x))g'(x).
\]
\end{theorem}
The formula can be expressed by saying that the derivative of a composition is ``the derivative of the outside   
times the derivative of the inside." It must be noted that the derivative of the outside is evaluated at the inside.  



\begin{example}[example 1]
Find $\dfrac{d}{dx}\left(\sqrt{x^2 + 9}\right)$.\\
The function $\sqrt{x^2 + 9}$ is a composition with inside function $g(x) = x^2 + 9$  and outside function $f(x) = \sqrt{x}$.
\[g(x) = x^2 + 9   \quad \text{and} \quad  f(x) = \sqrt{x}.\]


To find the derivative of $\sqrt{x^2 + 9}$ we use the chain rule, which requires us to find $f'(g(x))$ and $g'(x)$.
The second is simpler: $g'(x) = 2x$.
To find $f'(g(x))$ we must first find $f'(x)$, and then we substitute $g(x)$ for $x$. 
We have, 
\[
f'(x) = \frac{1}{2\sqrt{x}} \qquad \mbox{so,} \qquad f'(g(x)) = \frac{1}{2\sqrt{x^2 + 9}}.  
\]
Notice that the inside of the original function is now inside its derivative.
The chain rule now says that
\[
\frac{d}{dx}\left[\sqrt{x^2 + 9}\right] = f'(g(x))g'(x) = \frac{1}{2\sqrt{x^2 + 9}} \cdot 2x = \frac{x}{\sqrt{x^2 + 9}}.
\]

\end{example}



\begin{example}[example 1]
Find $\dfrac{d}{dx}\left[\sin(2x)\right]$.\\
The function $\sin(2x)$ is a composition with $g(x) = 2x$ inside and $f(x) = \sin(x)$ outside.
\[g(x) = 2x   \quad \text{and} \quad  f(x) = \sin(x).\]


To find the derivative of $\sin(2x)$ we need to use the chain rule.
We need both $f'(g(x))$ and $g'(x)$. The second is simpler so we start there. We have, $g'(x) = 2$.
To find $f'(g(x))$ we first find $f'(x)$ and then substitute $g(x)$ for $x$. 
We have, 
\[
f'(x) = \cos(x) \qquad \mbox{so,} \qquad f'(g(x)) = \cos(g(x)) = \cos(2x)
\]
We can now conclude that
\[
\frac{d}{dx}\left[\sin(2x)\right] = f'(g(x))g'(x) = \cos(2x) \cdot 2 = 2\cos(2x).
\]

\end{example}



\[g'(x) = 2; \quad \text{next} \]
\[f'(x) = \cos(x), \quad \text{so}\]
\[f'(g(x)) =\cos(g(x)) = \cos(2x).\]
By the chain rule,
\[h'(x) = f'(g(x))g'(x) = \cos(2x) \cdot 2 = 2\cos(2x).\]
We can also solve this problem using Leibniz notation.  In that case, we let $u = 2x$, and $y=\sin(2x)$.
Substituting $u$ for $2x$ in $y$, we can write 
\[
y = \sin(u),  \text{ where } \, u = 2x.
\]
Then the derivatives are
\[
\frac{dy}{du} = \cos(u)  \,\,\,\text{  and  }  \,\,\,\frac{du}{dx} = 2.
\]
Finally, the chain rule says to multiply these derivatives:
\[
\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = \cos(u) \cdot 2 = 2\cos(2x).
\]

\begin{example}[example 2]

Find $h'(x)$ if $h(x) = e^{5x}$.\\
We write $h(x)$ as a composition: $h(x)=f(g(x))$ with
\[
g(x) = 5x  \quad \text{and} \quad f(x) = e^x.
\]
To find $h'(x)$ we need $f'(g(x))$ and $g'(x)$. We have,
\[
g'(x) = 5; \quad \text{and} 
\]
\[
f'(x) = e^x, \quad \text{so}
\]
\[
f'(g(x)) =e^{g(x)} = e^{5x}.
\]
By the chain rule,
\[
h'(x) = f'(g(x))g'(x) = e^{5x} \cdot 5 = 5e^{5x}.
\]

\end{example}

\begin{example}[example 5]

Find $h'(x)$ if $h(x) = \sqrt{x^2 + 4}$.\\
We write $h(x)$ as a composition: $h(x)=f(g(x))$ with 
\[
g(x) = x^2 + 4  \quad \text{and} \quad  f(x) = \sqrt x.
\] 
To find $h'(x)$ we need $f'(g(x))$ and $g'(x)$.  We have,
\[
g'(x) = 2x; \quad \text{next} 
\]
\[
f'(x) = \frac{1}{2\sqrt x} , \quad \text{so}
\] 
\[
f'(g(x)) = \frac{1}{2\sqrt {g(x)}}=\frac{1}{2\sqrt {x^2 + 4}}.
\]
By the chain rule,
\begin{align*}
h'(x) &= f'(g(x))g'(x)\\
&= \frac{1}{2\sqrt {x^2 + 4}}\cdot 2x\\
&= \frac{2x}{2\sqrt {x^2 + 4}}\\
=\frac{x}{\sqrt {x^2 + 4}}.
\end{align*}
\end{example}

\begin{example}[example 13]
Find $h'(x)$ if $h(x) = \ln(\sin(x))$.\\
We write $h(x)$ as a composition: $h(x)=f(g(x))$ with 
\[g(x) = \sin(x)  \quad \text{and} \quad  f(x) = \ln(x).\]
 To find $h'(x)$ we need $f'(g(x))$ and $g'(x)$.  We have,
\[g'(x) = \cos(x); \quad \text{next} \]
\[f'(x) = \frac{1}{x}, \quad \text{so}\]
\[f'(g(x)) = \frac{1}{g(x)} = \frac{1}{\sin(x)}.\]
By the chain rule,
\[h'(x) = f'(g(x))g'(x) = \frac{1}{\sin(x)} \cdot \cos(x) = \frac{\cos(x)}{\sin(x)} = \cot(x).\]
\end{example}

We now create a general chain rule formula for the natural logarithm function.  
With $g(x)$ as the inside function, and we obtain the following formula for the derivative of $\ln(g(x))$:
\[
\frac{d}{dx} \left[\ln(g(x))\right] = \frac{1}{g(x)}\cdot g'(x) = \frac{g'(x)}{g(x)}.
\]


The above formula is both useful and important.  We will see it again in the Logarithmic Differentiation section.

\begin{example}[example 14]
 Find $\frac{d}{dx} \left[\ln(x^3 + 2)\right]$.\\
 
We use the above formula with $g(x) = x^3 + 2$ to get
\[
\frac{d}{dx} \left[\ln(x^3 + 2)\right] = \frac{3x^2}{x^3 + 2}.
\]
\end{example}

\section{Leibniz Notation for the Chain Rule}

The chain rule can also be expressed in Leibniz notation.  A composition $y = f(g(x))$ can be written using a chain of variables 
as: $y = f(u)$ where $u = g(x)$. The chain rule in this case is given by the elegant formula:
\[
\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx},
\]
In this format, the chain rule expresses the fact that ``rates of change multiply."  
To see this, consider a small change in the variable $x$  This produces a change in $u$ which in turn produces a change in $y$.
The chain rule states that the rate of change of $y$  with respect to $x$ is the rate of change of $u$ with 
respect to $x$ times the rate of change of $y$ with respect to $u$.
For example, if $u$ is changing three times faster than $x$ and $y$ is changing four times faster than $u$, then $y$ is changing $3 \cdot 4 = 12$ times faster than $x$.
